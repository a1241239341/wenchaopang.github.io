3.HTML常用标签

November 23, 2019

a 标签 超链接

a 标签

  • 属性
    • href 超链接
    • target 在哪个窗口打开链接
    • download 下载链接
    • rel=noopener 防止一个 bug
  • 作用
    • 跳转外部页面
    • 跳转内部锚点
    • 跳转到邮箱或电话等

例如：

<p>
  I'm creating a link to
  <a href="https://www.mozilla.org/en-US/">the Mozilla homepage</a>.
</p>

结果如下所示： I’m creating a link to the Mozilla homepage.

a 的 href 的取值

  • 网址
    • https://google.com
    • http://google.com
    • //google.com
  • 路径
    • /a/b/c 以及 a/b/c
    • index.html 以及./index.html
  • 伪协议
    • javascript:代码；
    • mailto：邮箱
    • tel：手机号
  • id
    • href=#xxx

a 的 target 的取值

  • 内置名字：
    • _blank 在空白页面打开
    • _top 在最顶层页面打开（有两个窗口）
    • _parent 在当前链接的页面的上一层打开
    • _self 在当前页面打开
  • 程序员命名：
    • window 的 name
    • iframe 的 name

a 的 download

  • 作用
    • 不是打开页面，而是下载页面

  • 问题

    • 不是所有浏览器都支持，手机浏览器可能不支持

table 标签 表格

语法

三元素

包含 table row 行 table date 数据 table head 表头

相关的样式

  • table-layout 包含 auto ，fixed
  • border-collapse 表格合并
  • border-spacing 表格空隙为零

img 标签 图片

  • 作用：发出 get 请求，展示一张图片
  • 属性：
    • alt 可选择的，可替换的
    • height 高度
    • width 宽度
    • src 地址
  • 事件
    • onload
    • onerror
  • 响应式
    • max-width:100% (手机页面)
  • 可替换元素：被问概率 30%
